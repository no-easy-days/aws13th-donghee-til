## list [ ]
여러 값을 순서대로 저장하는 상자(수정O)
```          
fruist = ["사과", "바나나", "딸기",]
//          0번     1번      2번
numbers = [1, 2, 3, 4, 5]
mixed = ["사과", 25, True] # 다른 종류의 값도 섞을 수 있다!
```
## list 수정하기
| 메서드 | 설명 | 사용 예시 |
| :--- | :--- | :--- |
| `append(x)` | 맨 뒤에 `x` 추가 | `list.append("새값")` |
| `insert(i, x)` | `i`번 위치에 `x` 끼워넣기 | `list.insert(0, "맨앞")` |
| `remove(x)` | `x` 값 삭제 (첫 번째 발견된 것만) | `list.remove("삭제할값")` |
| `pop()` | 마지막 값 꺼내기 (반환 및 삭제) | `last = list.pop()` |
| `sort()` | 오름차순 정렬 | `list.sort()` |
| `reverse()` | 순서 뒤집기 | `list.reverse()` |
| `len(list)` | 길이(아이템 개수) 확인 | `print(len(list))` |

## Dictionary { }
키을 이용하여 값을 저장하고 찾음
```
# 중괄호 {}와 콜론 :으로 만들어요
# {키: 값, 키: 값, ...}

student = {
    "이름": "김철수",
    "나이": 20,
    "전공": "컴퓨터공학"
}
```

딕셔너리 전체를 나타낼 때는 { ... }

그 안의 값 하나를 건드릴 때는 [ ... ]

## dictionary 수정하기
| 메서드 / 문법 | 설명 | 사용 예시 |
| :--- | :--- | :--- |
| `dict[key] = value` | 키-값 추가 또는 수정 | `contacts["친구"] = "010-0000"` |
| `update(dict2)` | 여러 키-값 한 번에 추가/수정 | `contacts.update({"a": 1, "b": 2})` |
| `get(key)` | 안전하게 값 가져오기 (에러 방지) | `contacts.get("엄마", "없음")` |
| `keys()` | 모든 **키(Key)**만 가져오기 | `list(contacts.keys())` |
| `values()` | 모든 **값(Value)**만 가져오기 | `list(contacts.values())` |
| `items()` | **키-값 쌍** 전체 가져오기 (반복문용) | `for k, v in contacts.items():` |
| `del dict[key]` | 특정 키로 삭제 | `del contacts["엄마"]` |
| `pop(key)` | 삭제하면서 값을 반환받기 | `removed = contacts.pop("친구")` |
| `clear()` | 전부 삭제 (초기화) | `contacts.clear()` |



## set { }
중복 및 순서가 없는 자료
```
# 중괄호 {}로 만들어요 (딕셔너리와 비슷하지만 콜론이 없어요)
numbers = {1, 2, 3, 4, 5}

# 중복은 자동으로 제거돼요!
duplicates = {1, 1, 2, 2, 3}
print(duplicates)  # {1, 2, 3}
```
set 중복 제거
```
# 중복이 있는 리스트
names = ["철수", "영희", "철수", "민수", "영희"]

# 세트로 바꾸면 중복 제거!
unique_names = set(names)
print(unique_names)  # {'철수', '영희', '민수'}

# 다시 리스트로 바꾸기
unique_list = list(unique_names)
print(unique_list)  # ['철수', '영희', '민수']
```

## f-string
```
name = "철수"
age = 20

# f-string 사용 (가장 쉬운 방법!)
print(f"이름: {name}, 나이: {age}세")
# 이름: 철수, 나이: 20세

# 계산도 가능해요
print(f"내년 나이: {age + 1}세")
# 내년 나이: 21세

```
