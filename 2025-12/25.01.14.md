## Python Type Hints
이 변수가 어떤 종류인지 단서를 남기는 것
타입 힌트는 강제하지 않는 메모임 하지만 fast api 하는게 좋음


```
# 타입 힌트 없음
name = "고동희"

# 타입 힌트 있음
name: str = "고동희"
```

```
# 함수 타입 힌트 없음
def greet(name):
    return name.  # 여기서 . 찍으면 자동완성이 안 뜸

# 함수 타입 힌트 있음
def greet(name: str):
    return name.  # 여기서 . 찍으면 upper(), lower() 등 str 메서드가 뜸

#리턴값도 힌트를 붙일 수 있음
def greet(name: str) ->str :
  return name
```

## 리스트
```
# 그냥 list라고만 쓰면
names: list = ["jeff", "철수", "영희"]
# 편집기는 names 안에 뭐가 들어있는지 모름

# 대괄호 안에 내용물 타입을 적으면
names: list[str] = ["jeff", "철수", "영희"]
# 편집기가 "아, 문자열 리스트구나" 이해함

```

## 딕셔너리
```
# 키는 문자열, 값은 정수
scores: dict[str, int] = {"jeff": 95, "철수": 88}
```

## 튜플
```
# 정확히 3개의 값 (문자열, 정수, 불리언)
user: tuple[str, int, bool] = ("jeff", 25, True)

튜플은 각 위치의 타입을 순서대로 적습니다.
```
## 선택적 값 표현하기
```
# 필수 값
name: str

# 선택 값 - 문자열이거나 없을 수 있음
nickname: str | None = None
```
