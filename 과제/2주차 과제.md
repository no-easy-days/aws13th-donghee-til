## 🔥 실습 8: 계산기 만들기 (심화)

두 숫자와 연산자를 입력받아 계산 결과를 출력하는 간단한 계산기를 만드세요.

- 덧셈, 뺄셈, 곱셈, 나눗셈, 몫, 나머지를 모두 지원
```
a,b = map(int,input("두 숫자를 입력하시오:").split())
Operator = input("연산자를 입력하시오: ")

if Operator == "+":
    print("덧셈:",(a + b))
elif Operator == "-":
    print("뺄셈:",a - b)
elif Operator == "*":
    print("곱셈:",a * b)
elif Operator == "/":
    print("나눗셈:",a / b)
elif Operator == "//":
    print("몫:",a // b)
elif Operator == "%":
    print("나머지:",a % b)
else:
    print("지원하지 않는 연산자입니다.")
    
```

## 🔥 실습 3-3: 3의 배수 찾기

```

    if number % 3 == 0:
        result.append(number)

print(result)
```
